# Installation sur Render.com (gratuit)

## 1. INSTALLATION LOCALE

```bash
# Clone ou télécharge les fichiers
# Crée un dossier avalon-app et mets tous les fichiers dedans

# Crée un virtual environment
python -m venv venv

# Active le venv
# Sur Windows:
venv\Scripts\activate
# Sur Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Lance l'app
python app.py
```

L'app run sur http://localhost:5000

---

## 2. DÉPLOYER SUR RENDER.COM (GRATUIT)

### Étape 1: Prépare les fichiers
Tes fichiers doivent être organisés comme ça:
```
avalon-app/
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
├── static/
│   ├── style.css
│   └── script.js
└── Procfile (créé à l'étape 2)
```

### Étape 2: Crée un fichier Procfile
Dans le dossier racine (avalon-app/), crée un fichier nommé `Procfile` (pas d'extension):

```
web: python app.py
```

### Étape 3: Push sur GitHub
```bash
# Initialize git repo (si pas déjà fait)
git init
git add .
git commit -m "Initial commit"

# Crée un repo sur GitHub et push
git remote add origin https://github.com/TON_USERNAME/avalon-app.git
git branch -M main
git push -u origin main
```

### Étape 4: Deploy sur Render
1. Va sur https://render.com
2. Sign up / Login
3. Click "New" → "Web Service"
4. Sélectionne ton repo GitHub
5. Configure:
   - **Name**: avalon-app
   - **Environment**: Python
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python app.py`
   - **Plan**: Free
6. Click "Create Web Service"

Render va builder et deploy l'app. Tu auras une URL du type:
`https://avalon-app-xxxxx.onrender.com`

---

## 3. UTILISER L'APP

### Game Master:
1. Clique "I'm Game Master"
2. Clique "Create Game" → Tu reçois un code (ex: ABC123)
3. Partage le code avec les joueurs
4. Sélectionne les rôles que tu veux jouer
5. Clique "Assign Roles"
6. Contrôle le jeu avec les boutons

### Joueurs:
1. Clique "Join Game"
2. Entre le room code et ton nom
3. Attends que le GM distribute les rôles
4. Vois ton rôle et vote pendant le jeu

---

## 4. TROUBLESHOOTING

**"Module not found" error:**
- Vérifie que le venv est activé
- Relance `pip install -r requirements.txt`

**Socket.IO not connecting:**
- Regarde la console du navigateur (F12 → Console)
- Vérifie que app.py run sans erreurs

**Heroku deploy issues:**
- Utilise Render à la place (c'est mieux maintenant)

## 6. NOTES POUR TOI

- Les rôles sont assignés aléatoirement mais respectent les règles de compatibilité
- Les infos spéciales (Merlin voit les espions, etc.) s'affichent 10 sec et disparaissent
- Le GM voit tout en temps réel (comptes de votes, résultats, etc.)
- Les joueurs ne voient que ce qu'ils doivent voir

C'est une base solide. La logique de jeu (débats, propositions, votes) est pas encore full automatisée — tu peux guider ça manuellement ou l'améliorer plus tard.
